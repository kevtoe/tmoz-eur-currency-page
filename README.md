## Getting Started with V0

Anyone can import this project into [v0.dev](https://v0.dev) and start building on top of it:

1. Click the **"Open in v0"** badge above, or paste the repo URL (`https://github.com/kevtoe/tmoz-eur-currency-page`) into v0's import dialog
2. V0 will import the codebase and let you make changes using AI prompts
3. When you're ready, use V0's GitHub integration to **create a new branch** and open a Pull Request back to this repo

### Alternative: Use as a Template

1. Click the **"Use this template"** button above (or on the GitHub repo page) to create your own copy
2. Import your new repo into V0
3. Push changes to branches on your own repo

## Running the code locally

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Tech Stack

- **React** with TypeScript
- **Vite** for build tooling
- **Tailwind CSS v4** for styling
- **Radix UI** + **MUI** components
- **Recharts** for data visualisation
- **Motion** for animations
